* The components used in this is a NodeMCU and a relay to give power to the high voltage appiances
* Intially you need to have an Arduino software
* Install the boards of NodeMCU
* Install the required library
* Create an account in the Adafruit IO and create the switches using the same names given in the code
* Create an accout in IFTTT and add the service of Adafruit IO and the Google Assistant
* Link those two services in IFTTT
* Adafruit will have one unique API key and the username(which you used for creation)
* Place those API key and username in the code
